# How to use DDos Golang

## Download
***Please Download Golang***

Then:

    git clone https://github.com/QuangHuwngg/DDos-Golang.git

Header.txt format:

    Accept: text/html
    User-agent: Wget
    Referer: http://google.com

## Usage

    cd DDos-Golang
    go build main.go
    main <url> <threads> <get/post> <seconds> <header.txt/nil>
